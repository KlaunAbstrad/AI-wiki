# ChatGPT 使用指南

## 一句话总结

**ChatGPT** 是 OpenAI 开发的 AI 对话助手，生态最完善，适合日常学习和工作。

## 版本

| 版本 | 价格 | 说明 |
|------|------|------|
| Free | 免费 | GPT-3.5，限额使用 GPT-4o |
| Plus | $20/月 | GPT-4/4o/4o mini，无限使用 |
| Pro | $200/月 | 最新模型抢先体验 |

## 核心功能

### 对话

- 自然语言提问
- 多轮对话保持上下文
- 支持中文

### 写作助手

- 撰写邮件、报告、文章
- 翻译语言
- 润色文案

### 编程辅助

- 写代码
- Debug
- 解释代码
- Code Review

### GPTs

定制化的 ChatGPT 版本，无需编码即可创建：
- 有特定知识的客服 bot
- 特定领域的助手
- 游戏助手

## 进阶使用

### Prompt 技巧

1. **明确角色**：你是一个 XXX 专家
2. **给出例子**：例如...
3. **分步骤**：第一步...第二步...
4. **指定格式**：用表格/列表/代码呈现

### 插件

Plus 用户可使用插件：
- 网页搜索
- 代码执行
- DALL-E 图像生成
