# Claude 使用指南

## 一句话总结

**Claude** 是 Anthropic 开发的 AI 助手，以长文本处理和强推理能力著称。

## 版本

| 版本 | 价格 | 说明 |
|------|------|------|
| Free | 免费 | Claude 3.5 Haiku，有使用限额 |
| Pro | $20/月 | Claude 3.5 Sonnet/Opus，无限使用 |
| Max | $100/月 | 最新模型，更高限额 |

## 核心优势

### 长文本理解

- 支持最多 200K token 上下文窗口
- 可以一次性阅读整本书籍、代码库、论文

### 推理能力

- 数学题、逻辑题表现优秀
- 复杂分析任务表现好

### 安全对齐

- Anthropic 强调 AI 安全，输出更有帮助且无害

## 平台

| 平台 | 链接 |
|------|------|
| Web | claude.ai |
| iOS | App Store 搜索 Claude |
| API | anthropic.com/api |

## Claude Code

命令行 AI 编程助手，适合开发者：
- 支持 Windows/Linux/macOS
- 可以自主操作文件系统、运行命令
- 适合 terminal-first 工作流
