# 欢迎来到 AI 学习小组 👋

这是一个由一群热爱学习 AI 的朋友共同建立的知识库。

## 🎯 我们的目标

- 记录 AI 学习过程中的知识点和踩坑经验
- 分享发现的优质工具、教程和论文
- 互相解答学习中的疑问
- 一起成长，从新手到进阶

## 📚 这个 Wiki 包含什么

| 板块 | 内容 |
|------|------|
| **知识库** | AI 核心概念、学习笔记、概念解析 |
| **资源** | 工具推荐、论文精选、学习链接 |
| **Q&A** | 常见问题解答、学习讨论 |
| **博客** | 学习心得、工具测评、教程分享 |

## 🚀 快速开始

1. **刚开始学习？** 从 [什么是 AI？](./categories/concepts/what-is-ai) 开始
2. **想找工具？** 查看 [AI 工具推荐](./resources/tools/ai-tools)
3. **有问题？** 在 [Q&A](./qa/getting-started) 搜索或提问
4. **想分享心得？** 写一篇 [博客文章](../blog)

## 💡 贡献指南

欢迎所有小组成员贡献内容！

### 如何添加内容

**添加新页面：**
```
docs/
├── categories/
│   └── concepts/
│       └── my-new-concept.md   ← 创建新文件
```

**页面基本格式：**
```markdown
---
id: my-concept
title: 我的新概念
---

# 我的新概念

这里写内容...
```

### 编辑现有内容

点击任意页面右上角的 **"编辑此页"** 链接即可直接在 GitHub 上修改。

## 📝 记录规范

- 概念名称：**中文（English）** 格式，方便中文用户理解
- 代码块：标注语言，如 `python`, `bash`
- 引用外部链接时说明来源
- 重要概念用 `:::info` 或 `:::tip` 标注

这是一个提示框，用 `:::tip` 包裹
