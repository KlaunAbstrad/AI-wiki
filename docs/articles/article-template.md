# 文章摘要模板

这是用于整理外部文章的标准格式。

## 模板

```markdown
---
id: 文章唯一标识（如 attention-is-all-you-need）
title: 文章标题
tags: [相关标签]
---

# 文章标题

## 📋 元信息

- **原文链接**：
- **作者**：
- **发布时间**：
- **阅读时长**：约 X 分钟
- **难度**：入门 / 进阶 / 专家

## 📝 一句话总结

用一句话概括这篇文章的核心内容。

## 🎯 核心观点

1. **第一点**：...
2. **第二点**：...
3. **第三点**：...

## 📊 关键图表/数据

（如果有重要的图表或数据，简要描述）

## 💡 对我的启发

（记录这篇文章对你学习 AI 的启发）

## 🔗 相关概念

- [相关概念 1](../categories/concepts/xxx)
- [相关概念 2](../categories/concepts/yyy)

## ❓ 存疑/待深入

（记录不懂的地方，后续研究）
```

## 使用方法

1. 发送文章链接给我
2. 我会帮你按照这个模板整理
3. 你可以直接复制到 `docs/articles/` 下使用
