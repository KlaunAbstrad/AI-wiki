# 关于我们

## 小组介绍

我们是一群 AI 学习者，有学生、设计师、程序员、产品经理...
我们因为对 AI 的兴趣聚在一起。

## 我们的学习方式

- 📚 共同维护这个 Wiki，记录学习成果
- 💬 微信群日常讨论和答疑
- 📝 定期分享学习心得和工具测评

## 参与贡献

欢迎所有成员贡献内容！

### 如何贡献

1. **直接编辑** — 点击任意页面右上角的"编辑此页"
2. **创建新页面** — 在对应目录下添加 `.md` 文件
3. **提出建议** — 在群里或提交 Issue

### 贡献原则

- 内容用**中文**撰写，概念词可附英文
- 保持内容**简洁易懂**，适合初学者
- 引用外部资源时注明**来源**
- 欢迎不同观点，理性讨论

## 联系方式

- **微信群**：加入方式见群公告
- **GitHub Issues**：提交问题或建议
