# 欢迎来到 AI 学习小组！

## 🎉 我们的 Wiki 上线了！

这是一个由一群 AI 学习者共同维护的知识库。

## 📚 内容规划

我们将在这里记录：

- **AI 核心概念** — 用简单中文解释复杂概念
- **工具推荐** — 我们在用、觉得好用的 AI 工具
- **学习心得** — 踩过的坑、获得的感悟
- **论文精选** — AI 领域的重要论文，附带中文解读

## 🚀 如何参与

### 编辑内容

点击任意页面右上角的 **"编辑此页"** 按钮，直接在 GitHub 上修改！

### 添加新内容

1. 在 `docs/` 目录下创建新的 `.md` 文件
2. 按照已有格式填写 frontmatter
3. 在 `sidebars.ts` 中注册新页面

### 提问和讨论

- 有问题？在 `docs/qa/` 中添加常见问题
- 有想法？提交 Issue 或 Pull Request

## 💡 内容规范

- **中文为主**：面向中文学习者
- **概念中英对照**：重要术语标注英文（如：大语言模型 / LLM）
- **简洁易懂**：适合初学者阅读
- **互相尊重**：欢迎不同观点，理性讨论

## 加入我们

如果你也想加入这个学习小组，联系群管理员获取邀请！
